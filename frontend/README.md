## Introduction

A Landing Page for Apollo Logistics Website using [Next.js](https://nextjs.org/).

## Installation

```bash
$ yarn install
```

## Running the app

Create `.env` file from `.env.example` and set environment variables before you run the service.

```bash
# Develop next
$ yarn dev

# Build next
$ yarn build

# Start next
$ yarn start

# Linter next
$ yarn lint
```
